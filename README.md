# ZSH_Install
ZSH一键安装优化脚本，装逼必备

## 安装方法

```bash
wget https://github.com/FanhuaCloud/ZSH_Install/archive/master.zip
unzip master.zip
cd ZSH_Install-master
./zsh.sh
```
